# studio-app

Releases do **Wann Studio** (montador de video profissional, by Wann Rodrigo).

Este repositorio guarda SOMENTE os instaladores das releases — o codigo-fonte vive em repositorio privado.

## Instalar

Baixe o `WannStudio_Setup_X.Y.exe` da release mais recente em **Releases** e execute. Os overlays do catalogo ja vem embutidos.

## Overlays extras (opcional)

Pra adicionar overlays proprios: use o botao "Adicionar externo" no card de Overlays do app, ou coloque arquivos `.mp4` em `%APPDATA%\WannStudio\overlays` — aparecem no catalogo na hora.
