# studio-app

Releases do **Wann Studio** (montador de video profissional, by Wann Rodrigo).

Este repositorio guarda SOMENTE os instaladores e assets das releases — o codigo-fonte vive em repositorio privado.

## Instalar

Baixe o `WannStudio_Setup_X.Y.exe` da release mais recente em **Releases** e execute.

## Overlays (opcional)

Baixe o `WannStudio_Overlays.zip` da release e extraia o conteudo em:

```
%APPDATA%\WannStudio\overlays
```

(cole o caminho na barra do Explorer; crie a pasta se nao existir). Os overlays aparecem no catalogo do app na hora — sem reinstalar.
